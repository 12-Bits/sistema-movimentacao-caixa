# Sistema de Movimentação de Caixa 💰

Projeto fullstack com:

- Backend: [NestJS](https://nestjs.com/) + TypeScript
- Frontend: [React](https://reactjs.org/) + Vite + TypeScript + Axios

---

## 📁 Estrutura do Projeto

sistema-movimentacao-caixa/ \n
├── backend/ → NestJS API \n
└── frontend/ → React com Vite \n

---

## 🚀 Como rodar o projeto

### Backend (NestJS)

```bash
cd backend
npm install
npm run start:dev


### Frontend (React + Vite)
cd frontend
npm install
npm run dev
