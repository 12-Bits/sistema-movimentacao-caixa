# Sistema de Movimentação de Caixa 💰

Projeto fullstack com:

- Backend: [NestJS](https://nestjs.com/) + TypeScript
- Frontend: [React](https://reactjs.org/) + Vite + TypeScript + Axios

---

Dependências:
Docker

---

## 📁 Estrutura do Projeto

sistema-movimentacao-caixa/ <br>
├── backend/ → NestJS API <br>
└── frontend/ → React com Vite <br>

---

## 🚀 Como rodar o projeto

### Backend (NestJS)

```bash
cd backend
npm install
npm run start:dev


### Frontend (React + Vite)
